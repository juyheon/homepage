# 🌐 프로젝트 설계 구상서

## 1. 개요
- 개인 글을 공유하는 블로
- 사용자 친화적인 UI, 댓글 소통 가능

## 2. 주요 기능
- 로그인/로그아웃
- 글 CRUD
- 댓글 작성 및 삭제
- 카테고리/태그 분류

## 3. 사용자 흐름
- 방문자 → 글 목록 → 상세 → 로그인 → 댓글
- 사용자 → 마이페이지 → 글 쓰기

## 4. 화면 정의
- 홈: 글 목록
- 글 상세: 제목, 본문, 댓글
- 로그인: 폼
- 마이페이지: 글 관리

## 5. 데이터베이스 구조
- User, Post, Comment 테이블 등

## 6. 기술 스택
- Front: React
- Back: Sanic (Python)
- DB: PostgreSQL
